# Sales Performance Dashboard

## Dashboard Preview

![Sales Dashboard](dashboard_preview.png)

## Overview
This project is an interactive sales performance dashboard built in Excel using PivotTables and slicers.

The dashboard provides analysis of:
- Total Revenue
- Total Units Sold
- Revenue by Product Category
- Date-based filtering

## Tools Used
- Microsoft Excel / WPS Office
- PivotTables
- Slicers
- KPI design principles
